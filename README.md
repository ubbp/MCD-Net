# MCD-Net
This repository is the offical implementation for "A Multiscale Vision-Text Collaborative Dual-Encoder for Referring RS Image Segmentation."

##  Installation

### Prerequisites
- Python ≥ 3.8
- PyTorch ≥ 1.10
- CUDA (for GPU training)
- Other dependencies listed in `requirements.txt`

### Setup
```bash
# Clone the repository
git clone https://github.com/your-username/MCD-Net.git
cd MCD-Net

# Create a virtual environment (recommended)
conda create -n mcdnet python=3.9
conda activate mcdnet

# Install dependencies
pip install -r requirements.txt
```

##  Dataset Preparation
We perform the experiments on two dataset including RefSegRS and RRSIS-D.
